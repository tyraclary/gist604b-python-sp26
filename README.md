**Module:** Assignment 4 – Python and GeoPandas

## Project Description
This project uses Python and GeoPandas to analyze spatial data and visualize results. It builds on PostGIS queries and converts them into Python workflows.

## Tools and Technologies
- Python
- GeoPandas
- Pandas
- Matplotlib

## What I Did
- Loaded spatial data into GeoPandas
- Performed data cleaning and filtering
- Conducted spatial analysis in Python
- Created visualizations using Matplotlib
- Exported results to GeoJSON

## How to View / Run
Run the Jupyter Notebook to view analysis and outputs.

- `notebooks/` – Jupyter notebooks
